DrawML

1. Data

You can upload data file in Data tab

Data input file name should be only English

Data input file content should be only Number

Data input file format should be only txt file

Data content format should follow form below

num_of_feature_x num_of_feature_y
contents....

Example

3 1
1 1 1 0
2 2 1 1
2 1 2 4
5 1 2 3
10 7 8 9

2. Experiment

You can make and run experiemnt in Experiment tab

Experiment shape should be acceptable, if not task will fail while running

Experiment input data should be acceptable, if not task will fail while running

You must click Save button or Run button

You can see state of experiment on footer-bar

3. Test

You can run Trained Model in Test tab

Trained model state will be shown at the detail page

Input data of trained Model should be acceptable, if not task will fail while running

You must click Clear status button after test was ended

4. Log

Trained and tested result are here Log tab

You can see the result log and edit file name in detail page
